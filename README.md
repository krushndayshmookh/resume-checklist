# Resume Review Checklist

A streamlined, single-page application for conducting thorough resume reviews with structured feedback.

## Features

- **10 Review Sections**: Meta, Presentation, Summary, Education, Skills, Projects, Experience, Achievements, Certificates, and Communication
- **Progressive Unlocking**: Sections unlock as you complete previous ones
- **Auto-save**: Form data automatically saved to browser's localStorage
- **Google Sheets Integration**: Submit reviews directly to Google Sheets
- **Export Options**: Download reviews as JSON

## Usage

1. For Local, open `index.html` in your browser
2. Enter student details and reviewer name
3. Complete each section with checkboxes and ratings
4. Provide feedback comments where needed
5. Submit to Google Sheets or download as JSON
